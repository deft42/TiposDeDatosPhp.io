Variables y tipos de datos.

echo

Es una construcción que se utiliza para imprimir o mostrar contenido en la salida. Básicamente, te permite mostrar texto, variables u otros datos en la página web que estás generando.

Sintaxis basica

// Escribirmos la construccion y seguido entre comillas, insertamos
// la cadena de texto que queramos desplegar.
echo "Mensaje que queramos mostrar en pantalla";

// Tambien tenemos la posibilidad de agregar variables dentro de la cadena de texto
// Declaramos una variable con el contenido dentro de ella.
$nombre = "Juan";
// Imprimimos en pantalla un mensaje e insertamos la variable.
echo "Mi nombre es $nombre.";

// La construccion echo, tiene la capacidad y facilidad de renderizar elementos html
// es decir que podemos agregar html dentro de la cadena de texto. Afectando el
// texto que la contiene.
echo "<h1>¡Bienvenido!</h1>";

// Por ultimo, podemos concatener texto junto con las variables. Semejante
// a javascript, en php la manera de concatenar las cadenas de texto es la 
// siguiente.
$edad = 25;

// 1.- Añadimos y separamos por partes el texto que queremos desplegar.
// 2.- eguido colocamos punto "." en conjunto con la variable y repetir
		// el paso anterior.
echo "Tengo " . $edad . " años.";

Definicion de variable.

Una variable es un contenedor para almacenar información, como números, texto o cualquier otro tipo de datos.

Declaración de variables.

// Una variable se compone de 3 partes.
// 1.- La primera parte esta representada con un signo de dolar "$", esto le dice a php
//     estas por inicializar la declaración de una variable.

// 2.- La segunda parte se compone del nombre ingresado por el usuario.

$variable = "Contenido de la variable";

		// en caso de que la variable contenga mas de una palabra, se sugiere usar el
		// metodo llamado "camelCase" el cual consiste en escribir la primera palabra 
		// en miusculas y las siguientes con la inicial mayuscula.

$nombreDeLaVariable ="Esta es una variable usando camelCase";

		// o tambien esta la alternativa de separar una variable por palabras compuestas
		// con guion bajo "_", por ejemplo:

$nombre_de_la_variable = "Otra variable declarada pero con guion bajo";

Enteros (Integer o int).

// Representa números enteros, positivos o negativos, sin decimales
$edad = 25;

<?php 
// Ejemplo de tipo de dato entero.

/*
    Definimos las variables que vamos a ocupar, 
    en este caso, seran dos por que realizaremos
    el calculo de dos numeros con operaciones 
    basicas.

    Dichos valores, seran de tipo entenro, es decir;
    Que los numeros no tienen punto decimal.
*/

// Variable 1
$numero_1 = 10;
// Variable 2
$numero_2 = 5;

/* 
    --------------------------------------------
    PROCESO DE DATOS Y OPERACIONES MATEMATICAS
    --------------------------------------------
*/

// Operacion 1.- Suma.
/*
    Creamos una variable que va a contener 
    la suma de los dos numeros. En este caso
    llamaremos a la variable "operacion suma"
    usando el metodo "camelCase".
*/ 

// Declaración de la nueva variable "operacion suma".
$operacionSuma = $numero_1 + $numero_2;
echo "El resultado es: $operacionSuma<br>";

// Operacion 2.- Resta.
/*
    Para la siguiente operacion, realizaremos
    realizaremos exactamente el mismo proceso.
    Crearemos una variable que contendra la 
    operacion de los dos varlores.
*/ 

// Declaración de la nueva variable "operacion resta".
$operacionResta = $numero_1 - $numero_2;
echo "El resultado es: $operacionResta<br>";

// Operacion 3.- Multiplicacion.
// Declaración de la nueva variable "operacion multiplicación".
$operacionMultiplicacion = $numero_1 * $numero_2;
echo "El resultado es: $operacionMultiplicacion<br>";

// Operacion 4.- Division.
// Declaración de la nueva variable "operacion division".

if($numero_2 != 0) {
    $operacionDivision = $numero_1 / $numero_2;
    echo "El resultado es: $operacionDivision<br>";
} else {
    echo "No se puede divir por 0";
}

?>

/*
    EJEMPLO PRACTICO DE TIPO DE DATO ARRAYS.
*/

/*
    Los tipos de datos arrays o arreglos,
    es en realidad un mapa ordenado. Un mapa
    es un tipo de datos que asocia valores con claves.

    Dentro de estos mapas nosotros podemos crear:
        - lista (vector)
        - tabla asociativa (tabla hash)
        - diccionario, colección, pila, cola.
*/

/*
    -----------------------------------------
    INICIO DEL PROCESO
    -----------------------------------------
*/
// Un arreglo o array, esta compueso por su variable y
// lista de datos que va a contener con su respectiva posicion.

// Creamos un array o arreglo de nombres
$nombres = ["Juan", "María", "Carlos", "Ana"];
//

// Mostrar los nombres
echo "Nombres: ";
// "foreach" es un modo sencillo de iterar sobre arrays. foreach funciona sólo sobre arrays y objetos.
// foreach = "por cada uno de"
// "as" es una manera de iterar dentro del forwach
foreach ($nombres as $nombre) { 
// Por cada uno de los nombres que este en la variable "nombres" iteralos y guardalos en la variable "nombre".
    echo $nombre . ", "; // Imprime y concatena los nombres de la nueva variable.
}

?>

<?php
// Definir una clase Person
class Person {
    public $nombre;
    public $edad;
    public $genero;

    // Constructor
    public function __construct($nombre, $edad, $genero) {
        $this->nombre = $nombre;
        $this->edad = $edad;
        $this->genero = $genero;
    }

    // Método para mostrar información de la persona
    public function mostrarInformacion() {
        echo "Nombre: $this->nombre<br>";
        echo "Edad: $this->edad años<br>";
        echo "Género: $this->genero<br>";
    }
}

// Crear una instancia de la clase Person
$persona1 = new Person("Ana", 25, "Femenino");

// Mostrar información de la persona
$persona1->mostrarInformacion();
?>
